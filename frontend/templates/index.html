Welcome to EyeOfCharacteristics

Simple tips:

  • Use this path for a POST-Request (image) to EOC /analyze-image

  • curl -i -H "Content-Type: application/json" -X POST -d '{"image":{YOUR-FILE}' https://{THIS-HOST}:443/analyze-image

  • It has to be a .jpg file

  • Use this path for a POST-Request (video) to EOC /analyze-video

  • curl -i -H "Content-Type: application/json" -X POST -d '{"image":{YOUR-FILE}' https://{THIS-HOST}:443/analyze-video

  • It has to be a .mov file

  • It is easier to use the EOC Vue.js Frontend, than the cURL commands

  • For more information: README.md

  • Vue.js Frontend: EyeOfCharacteristics
